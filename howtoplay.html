How to record scores

The following keys can be used in the input box to score tiles, with some "hotkeys" if you don't want to press the suit/concealment buttons.

Suited tiles

suited tiles simply use the tile's number, with the following hotkey letters anywhere in the pattern to set the suit:

  • - number 1-9 plus "c" for characters or the button.
  • - number 1-9 plus "b" for bamboo or the button.
  • - number 1-9 plus "d" for dots or the button.

Honour tiles

winds:

  • - east: "e"
  • - south: "s"
  • - west: "w"
  • - north: "n"

dragons:

  • - green dragon: "f"
  • - red dragon: "c"
  • - white dragon: "p"

Marking tiles as concealed

Any pattern can be marked as concealed either by typing "!" in the input field, or pressing the button on the far right of a set.

Special concealment for kongs

If a player draws a fourth tile of an already revealed pung, simply add that tile to the already recorded pung. Likewise, if a kong is played fully concealed, record it normally. However, if a player claims a tile from someone's discard to form a kong, they already had a concealed pung in their hand.
